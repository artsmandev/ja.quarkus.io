命令型とリアクティブ型の統合

アプリケーション開発時に、慣れ親しんだ命令型コードと、グリアクティブ型コードを組み合わせることが出来ます。 詳細.

命令型
@Inject
SayService say;

@GET
@Produces(MediaType.TEXT_PLAIN)
public String hello() {
    return say.hello();
}
リアクティブ型
@Channel("kafka") Multi<String> events;

@GET
@Produces(MediaType.SERVER_SENT_EVENTS)
public Multi<String> events() {
    return events;
}
